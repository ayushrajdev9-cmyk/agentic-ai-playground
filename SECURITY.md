# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please do NOT file a public issue.

Instead, send a description of the issue to security@agentic-ai-playground.dev.

We will acknowledge receipt within 48 hours and provide a timeline for resolution.

## Best Practices

- Never commit API keys or tokens
- Use environment variables for secrets
- Run code execution tools in sandboxed environments
- Review tool permissions before enabling
- Keep dependencies updated
